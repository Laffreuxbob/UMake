U-MAKE

Projet U-Make RPG

Jeu rpg (en 2d ?) en C# ou Java, pour se chauffer sur un langage cool pour dev, et sympa sur le cv.

But

Etre riche assez vite

CDC de base

Contenu de jeu, gameplay :

  • 2 classes (guerrier et mage, pour commencer)
  • 3 niveaux / donjons (changement de map)
  • 1 boss
  • 3 items (potion, épée, monture)

Fonctionnement :

  • 1 système de sauvegarde => BDD ?
  • Multijoueurs (à deux su un réseau local, ça peut être intéressant)
  • Niveau, expérience ?

Techno

  • Java
  • C#
  • MongoDB / MySQL / autres

Les personnages :

class character(){
    name, hp, speed, scope
}

class wizard extends character(){
    super("wizardName", 35, 1, 10), power;
}

class warrior extends character(){
    super("warriorName", 50, 1, 2), strength;
}

Des fonctions diverses :

function addHorse(newHorse){
    if (this.horse){  // If this.stuff.contains(horse)
        this.speed += newHorse.speed;
    }
}

function addWeapon(newWeapon){
    this.power/strength += newWeapon.addStat
}

function heal(){
    if (this.stuff.contains(curePotion){
        this.hp += 20;
        this.stuff.remove(curePotion);
    }
}
